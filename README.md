# Drop Number Game

Drop Number Game is a Java-based desktop application that demonstrates the practical use of custom linked list data structures within an interactive number game.

The project combines data structures, object-oriented programming, and graphical user interface development. Game data and number sequences are managed using custom node-based structures instead of relying entirely on Java's built-in collection classes.

## Features

- Interactive desktop number game
- Java Swing graphical user interface
- Custom linked list implementation
- Node-based dynamic data management
- Dynamic number sequence management
- Event-driven user interactions
- Object-oriented and modular application structure

## Technologies

- Java
- Java Swing
- Apache Ant

- Git and GitHub

## Core Components

### GameFrame

Manages the graphical user interface, game screen, and user interactions using Java Swing.

### MultiLinkedList

Implements the custom linked list structure used to organize and manage game-related data dynamically.

### NextNumbersList

Manages the sequence of upcoming numbers and supports the number flow used during gameplay.

### NodeClass

Defines the node structure used by the custom linked list implementations.

## Data Structure Implementation

The project focuses on applying linked list concepts directly within game logic. Custom node-based structures are used for dynamic data organization, linked list traversal, and number sequence management.

This approach demonstrates how fundamental data structures can be integrated into a functional desktop application.

## Installation and Usage

Clone the repository:

```bash
git clone https://github.com/rumeysakocakk/drop-number-game.git
```

Navigate to the project directory:

```bash
cd drop-number-game
```

Open the project in NetBeans IDE, build the project, and run the application.
